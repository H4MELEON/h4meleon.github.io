Гипертекст - это структурированный текст, созданный с использованием языка разметки, потенциально имеет гиперссылки.

HTML (HyperText Markup Language) - фзык разметки гипертекста.

    Способы создания HTML-документа:
  1. Испоьзуя простой текстовый редактор
  2. При помощи редактора кода
  3. С помощью визуального web-редактора

Тэг - управляющий символ языка разметки. Существуют одинарные тэги и тэги-контейнеры. Тэги-контейнеры состоят из открывающего и закрывающего тэга (<h1>Заголовок первого уровня</h1>). Одинарный тэги состоят только из открывающего тэга (<br> - осуществляет перенос строки)

Атрибут - свойство тэга, записывается только в открывающем тэге (<тэг атрибут="значение">). Одним из важнейших атрибутов является class, данный атрибут позволяет присвоить любому элементу класс, которому можно задавать определённый свойства через каскадные таблицы стилей (CSS).

Для оформления документа существует ряд базовых тэгов, которые необхожимы при создании каждого HTML-документа:
<html> //Указывает браузеру, что этот файл необходимо читать, как HTML-документ
<head> //Содержит информацию для поисковых систем
<title>Заголовок страницы</title>
</head>
<body> //Содержит информацию, отображаемую пользователям
</body>
</html>
